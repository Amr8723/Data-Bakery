## 🧁 The Concept

**Data Bakery** transforms tedious data workflows into a delightful baking experience. No more dry spreadsheets — knead your data, let it rise with simulations, and serve up beautiful insights.

| Bakery Term | Data Meaning |
|-------------|--------------|
| 🥣 **Prep** | Upload & profile datasets |
| 🔥 **Oven** | Transform with recipes |
| ⏰ **Timewarp** | Simulate future trends |
| 🧵 **DataLoom** | Weave visualizations |
| 📊 **Report** | AI-assisted insights |

---

## ✨ Features

### 🥣 Prep View
**Upload → Profile → Check Freshness**
- Schema auto-detection
- Missing value scanning
- Statistical summary

### 🔥 Oven View
**Apply Recipes to Transform Data**
- Filter & clean
- Merge datasets
- Aggregate & reshape

### ⏰ Timewarp View
**Simulate Future Trends**
| Ingredient | Parameter |
|------------|-----------|
| 🧪 Yeast | Variance (randomness) |
| 🍬 Sugar | Trend (growth direction) |

### 🧵 DataLoom & Exploration
**Interactive Visualizations**
- Scatter plots
- Line charts
- Bar charts
- Heatmaps

### 📊 Report View
**AI-Generated Insights**
Powered by Gemini
